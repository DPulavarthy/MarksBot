Home Page

MarksBot is a Discord moderation and fun bot it has plenty of commands to keep you busy.

If you need support you are welcome to join our Discord server you can also invite MarksBot to your server by clicking here. If you have suggestions you can DM the bot or join the support server and post your suggestion in there. Want to read the FAQ? click here

Admin / Moderator Commands
Name Description Example
m.addemoji [url] [name] Let's you add an emoji to the server via a URL
m.allwarns Shows up to 25 warnings
m.ban @user [reason] Bans the tagged user and logs the reason (also send them a DM if they have DM's open)
m.banlist Display the list of users who are banned from the server
m.banrole [role] Bans ALL users within the selected role enable/disable with m.optout m.banrole members
m.blacklist @user Prevents the tagged user from using MarksBot commands
m.case [number] View a specific warnings case m.case 1
m.check Checks to see if any users have invite links in their status message
m.co [#channel] or [id] View the permission overwrites for the channel
m.filter This is used to set your custom word filter and MarksBot will remove messages that contain these words, use m.filter to see your options
m.kick @user [reason] Kicks the tagged user from the server and logs the reason (also send them a DM if they have DM's open)
m.kickrole [role] Kicks ALL users within the selected role enable/disable with m.optout m.kickrole members
m.mute @user Mutes a user so they cannot speak, if no time is set then it defaults to a 1 hour mute m.mute @markawes 1d spamming
m.prune [number 1-100] Bulk delete messages up to 100 or m.prune @user bulk delete messages for the tagged user
m.resetlb Resets the server leaderboards for m.spinner
m.resetpilots Resets the server leaderboards for m.plane
m.server Locks down the server so only staff can chat, please run m.help server for more information
m.softban @user @user etc Bans and immediately unbans the tagged user(s), removing 7 days of their messages
m.starboard Used to control starboard options, such as channel, required stars etc.
m.timeout @user/bot Used to timeout a user or bot for up to 1 week m.timeout @markawes 2d
m.toggle Enabled/Disables all commands or a specific command in the server Only the server owner or users with the ADMINISTRATOR permission can use this command m.toggle ping
m.topinvites Shows you the current invite leaderboard with the most uses This command is locked to mods because it shows invite codes & some servers are private communities and do not want members creating invites - Use it in a private channel as it shows invite codes
m.unblacklist @user Allows a blacklisted user to use MarksBot commands again
m.unmute @user Unmutes a muted user
m.votekick @user Provides a poll where users can vote to kick the selected user, if the poll gets the requireed amount of vote the user is kicked
m.warn @user or [id] [warn reason] Issues the tagged user with a warning and logs it m.warn @markawes spam
m.warn- [case id] Remove the warning from a user (Case number required) m.warn- 1
m.warns @user or [id] View the users warnings m.warns @markawes
